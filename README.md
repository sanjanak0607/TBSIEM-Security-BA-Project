# TBSIEM Security Dashboard

## Project Overview
Conducted end-to-end business analysis for TechBridge's 
cybersecurity product TBSIEM (Threat-Based Security 
Information and Event Management).

## What I Did
- Researched the SIEM market and competitive landscape
- Created a structured Business Requirements Document (BRD)
- Built a professional product overview presentation
- Designed a Security Operations Center (SOC) Dashboard 
  wireframe in Figma

## Deliverables
- Business Requirements Document (BRD)
- Product Presentation that included Overview & Value Proposition
- Security Dashboard Wireframe (Figma)

## Skills Demonstrated
Business Requirements Documentation, Stakeholder Analysis,
Wireframing, Market Research, Presentation Skills
